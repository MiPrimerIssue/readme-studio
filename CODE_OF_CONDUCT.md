# 💙 Código de conducta

## 🌍 Nuestro compromiso

Queremos que este proyecto sea un espacio abierto, respetuoso, seguro y acogedor para todas las personas que participen.

Este repositorio también está pensado como entorno de aprendizaje, así que cuidar la forma en la que colaboramos es tan importante como el propio código.

---

## ✅ Comportamiento esperado

Esperamos que quienes participen:

- se comuniquen con respeto
- den feedback constructivo
- tengan paciencia con quienes están empezando
- colaboren con actitud positiva
- ayuden a crear un entorno inclusivo

---

## ❌ Comportamientos no aceptables

No se tolerarán:

- insultos o ataques personales
- lenguaje ofensivo o agresivo
- discriminación de cualquier tipo
- acoso o intimidación
- conductas que dificulten la participación de otras personas

---

## 🛡 Aplicación

Las personas mantenedoras del proyecto podrán intervenir si detectan comportamientos inadecuados y tomar las medidas necesarias para proteger el entorno de colaboración.

---

## 🎯 Objetivo

Este proyecto existe para aprender, colaborar y construir experiencia real en open source. Queremos que cualquier persona pueda participar sintiéndose bienvenida.
