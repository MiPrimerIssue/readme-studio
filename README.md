# 🚀 README Studio

**README Studio** es una aplicación web open source diseñada para Herramienta open source para generar README profesionales de GitHub mediante bloques guiados y vista previa en tiempo real.

Este proyecto forma parte del ecosistema **MiPrimerIssue**, una iniciativa orientada a que desarrolladores junior hispanohablantes puedan contribuir a proyectos reales mediante tareas guiadas, progresivas y útiles para su aprendizaje.

---

## ✨ ¿Qué hace este proyecto?

README Studio permite:

- Crear README desde un formulario guiado
- Vista previa en tiempo real
- Plantillas por tipo de proyecto
- Copiar y exportar Markdown
- Aprender estructura de documentación profesional

---

## 🎯 Objetivo del proyecto

Generar README profesionales para proyectos de GitHub con secciones guiadas, vista previa y exportación en Markdown.

---

## 🧰 Tecnologías utilizadas

Este proyecto utiliza:

- **Next.js**
- **TypeScript**
- **Tailwind CSS**
- **Supabase**

---

## ⚙️ Instalación

Clona el repositorio:

```bash
git clone https://github.com/MiPrimerIssue/readme-studio.git
cd readme-studio
```

Instala dependencias:

```bash
npm install
```

Copia el archivo de entorno:

```bash
cp .env.example .env.local
```

Ejecuta el proyecto en local:

```bash
npm run dev
```

Abre en tu navegador:

```text
http://localhost:3000
```

## 📁 Estructura del proyecto

```text
docs/           → documentación general del proyecto
tasks/          → tareas organizadas por nivel
.github/        → plantillas de Pull Requests e Issues
src/            → código fuente de la aplicación
```

---

## 🤝 Cómo contribuir

1. Lee `CONTRIBUTING.md`
2. Revisa la carpeta `tasks/`
3. Solicita una tarea desde MiPrimerIssue
4. Espera a que se te asigne
5. Trabaja en tu solución
6. Abre una Pull Request
