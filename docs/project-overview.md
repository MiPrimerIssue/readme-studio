# 📘 Descripción general del proyecto

## 🚀 ¿Qué es README Studio?

README Studio es una aplicación web pensada para generar README profesionales para proyectos de GitHub con secciones guiadas, vista previa y exportación en Markdown.

El proyecto busca ser útil tanto para personas que necesitan una herramienta real como para juniors que quieren aprender contribuyendo a una aplicación concreta.

---

## ❓ Problema que resuelve

Herramienta open source para generar README profesionales de GitHub mediante bloques guiados y vista previa en tiempo real.

---

## 👥 Público objetivo

Este proyecto está orientado a:

- desarrolladores junior
- estudiantes de programación
- personas que quieren practicar con un producto real
- contribuidores que quieren iniciarse en open source

---

## 🧩 Funcionalidades principales

- Crear README desde un formulario guiado
- Vista previa en tiempo real
- Plantillas por tipo de proyecto
- Copiar y exportar Markdown
- Aprender estructura de documentación profesional
---

## 🎯 Alcance inicial

La primera versión del proyecto se centrará en una experiencia clara, visual y fácil de ampliar mediante tareas pequeñas y bien definidas.
