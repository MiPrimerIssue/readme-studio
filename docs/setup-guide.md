# 🛠 Guía de configuración

## ✅ Requisitos previos

Antes de empezar, asegúrate de tener instalado:

- **Git**
- **Node.js**
- **npm** o **pnpm**

---

## 📦 Instalación

Clona el repositorio:

```bash
git clone https://github.com/MiPrimerIssue/readme-studio.git
cd readme-studio
```

Instala las dependencias:

```bash
npm install
```

Copia el archivo de entorno:

```bash
cp .env.example .env.local
```

Ejecuta el proyecto en local:

```bash
npm run dev
```

---

## 🌐 Acceso local

Abre el proyecto en tu navegador:

```text
http://localhost:3000
```

---

## 🧭 Recomendaciones para contribuir

Antes de empezar una tarea:

- lee bien su descripción en `tasks/`
- revisa los archivos sugeridos
- mantén tus cambios centrados en una sola tarea
- evita modificar partes no relacionadas
