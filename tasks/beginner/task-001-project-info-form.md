# 📝 Task 001 - Project Info Form

## Nivel

Beginner

---

## 🎯 Objetivo

Crear un formulario reutilizable para capturar la información básica del proyecto.

---

## 🧠 Contexto

Esta tarea forma parte de la primera versión del proyecto y está pensada para construir una base visual clara, reutilizable y fácil de mantener.

---

## ✅ Requisitos

- Crear un componente llamado `ProjectInfoForm`
- Debe incluir campos para nombre del proyecto, descripción corta y stack
- Debe ser reutilizable
- Debe funcionar bien en móvil y escritorio
- Debe seguir el estilo visual general del proyecto

---

## 📁 Archivos sugeridos

```text
src/components/project-info-form.tsx
src/app/page.tsx
```

---

## ✔️ Criterios de aceptación

- El formulario renderiza correctamente
- Los campos se pueden completar y actualizar
- El componente es reutilizable
- No hay errores de TypeScript
- El cambio está centrado solo en esta tarea

---

## ✨ Extra opcional

- Añadir contador de caracteres a la descripción
- Mostrar ayuda visual bajo cada campo
