# 📭 Task 003 - Empty Template State

## Nivel

Beginner

---

## 🎯 Objetivo

Crear un estado vacío para cuando el usuario todavía no haya completado ningún dato del README.

---

## 🧠 Contexto

Esta tarea forma parte de la primera versión del proyecto y está pensada para construir una base visual clara, reutilizable y fácil de mantener.

---

## ✅ Requisitos

- Crear un componente para estado vacío
- Debe incluir mensaje claro y amigable
- Puede incluir icono o ilustración simple
- Debe ser reutilizable
- Debe mantener coherencia visual con el proyecto

---

## 📁 Archivos sugeridos

```text
src/components/empty-template-state.tsx
src/app/page.tsx
```

---

## ✔️ Criterios de aceptación

- El estado vacío se ve limpio
- El mensaje guía al usuario
- Se adapta bien a móvil y escritorio
- Es reutilizable
- El cambio está centrado solo en esta tarea

---

## ✨ Extra opcional

- Añadir CTA para empezar a completar el formulario
