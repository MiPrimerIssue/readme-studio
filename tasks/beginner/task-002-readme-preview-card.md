# 👀 Task 002 - README Preview Panel

## Nivel

Beginner

---

## 🎯 Objetivo

Crear un panel visual donde se muestre una vista previa del README generado.

---

## 🧠 Contexto

Esta tarea forma parte de la primera versión del proyecto y está pensada para construir una base visual clara, reutilizable y fácil de mantener.

---

## ✅ Requisitos

- Crear un componente llamado `ReadmePreview`
- Debe mostrar contenido en formato visual legible
- Debe manejar estados vacíos
- Debe adaptarse bien a diferentes tamaños de pantalla
- Debe encajar con una UI limpia y moderna

---

## 📁 Archivos sugeridos

```text
src/components/readme-preview.tsx
src/app/page.tsx
```

---

## ✔️ Criterios de aceptación

- La vista previa se muestra correctamente
- Tiene jerarquía visual clara
- Se ve bien en escritorio y móvil
- Es fácil de reutilizar
- El cambio está centrado solo en esta tarea

---

## ✨ Extra opcional

- Añadir botón de copiar contenido
- Añadir cabecera con nombre del proyecto
