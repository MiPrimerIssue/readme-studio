# 🧩 Task 101 - Template Selector

## Nivel

Intermediate

---

## 🎯 Objetivo

Implementar un selector de plantillas de README según el tipo de proyecto.

---

## 🧠 Contexto

Esta tarea amplía la funcionalidad base del proyecto y requiere integrar bien la solución con la navegación o el flujo principal de la aplicación.

---

## ✅ Requisitos

- Crear un selector con varias plantillas
- Debe permitir cambiar entre distintas opciones
- La selección debe actualizar la vista previa
- Debe integrarse bien con el formulario principal
- Debe mantener una experiencia simple y clara

---

## 📁 Archivos sugeridos

```text
src/components/template-selector.tsx
src/app/page.tsx
```

---

## ✔️ Criterios de aceptación

- El selector funciona correctamente
- La plantilla elegida actualiza la vista previa
- La solución es clara y mantenible
- Se puede volver a la opción inicial
- El cambio está centrado solo en esta tarea

---

## ✨ Extra opcional

- Añadir estilos activos a la plantilla seleccionada
- Guardar la selección en estado global
