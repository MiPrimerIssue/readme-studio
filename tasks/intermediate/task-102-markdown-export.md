# 📤 Task 102 - Markdown Export

## Nivel

Intermediate

---

## 🎯 Objetivo

Implementar la exportación del README generado en formato Markdown.

---

## 🧠 Contexto

Esta tarea amplía la funcionalidad base del proyecto y requiere integrar bien la solución con la navegación o el flujo principal de la aplicación.

---

## ✅ Requisitos

- Crear la lógica para transformar los datos en Markdown
- Añadir un botón para exportar o copiar
- La salida debe ser limpia y utilizable
- Debe integrarse con la información del formulario
- Debe ser fácil de mantener

---

## 📁 Archivos sugeridos

```text
src/components/export-markdown-button.tsx
src/lib/generate-markdown.ts
```

---

## ✔️ Criterios de aceptación

- El Markdown generado es válido
- El usuario puede copiar o exportar el contenido
- La solución está bien organizada
- No rompe otras partes de la aplicación
- El cambio está centrado solo en esta tarea

---

## ✨ Extra opcional

- Añadir feedback visual al copiar
